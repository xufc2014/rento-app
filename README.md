# 房东通（Rento）

> 极简出租屋管理助手 — 纯前端、零后端、离线可用

房东通是一款基于 uni-app 开发的跨平台出租屋管理应用，覆盖**楼栋房间管理、租客管理、合同签订、水电抄表、账单生成、退租结算**全流程，所有数据存储在本地设备，无需后端服务器。

---

## 功能矩阵

| 模块 | 功能 |
|------|------|
| 🏢 **楼栋管理** | 添加楼栋、批量生成房间、户型配置 |
| 🚪 **房间管理** | 按楼栋/楼层分组、状态流转（空置→已预订→已入住→退租中）、批量操作 |
| 👤 **租客管理** | 添加/搜索/编辑租客、黑名单标记 |
| 📝 **合同管理** | 6步向导签订、押金配置、初始水电底读、到期提醒（30/15/3天） |
| ⚡ **水电抄表** | 按月逐间录入（水/电/气）、历史比较、异常告警、拍照存证、逐项分别录入 |
| 📊 **抄表管理** | 按月筛选所有抄表记录、支持修改并级联更新后续记录 |
| 💰 **账单管理** | 批量生成月账单、费用明细（水/电/气/网/卫生/管理费）、减免、导出 CSV |
| 🔄 **退租结算** | 3步流程：确认信息→抄表→结算、自动计算应退押金、归档合同 |
| 📜 **操作日志** | 全程操作记录（含楼栋+房间号详细信息） |
| 💾 **数据备份** | 支持 JSON 本地导出/导入、生成账单自动备份 |

## 技术栈

| 层级 | 技术 |
|------|------|
| 框架 | uni-app + Vue 3 Composition API |
| 平台 | H5 / Android / iOS / 微信小程序 |
| 数据存储 | uni.storage（9个集合，纯客户端本地存储） |
| 邮件备份 | Resend REST API |
| 服务端（可选）| Vercel Serverless Function |

## 典型工作流

```
添加楼栋 → 批量生成房间 → 添加租客 → 签订合同（含初始底读）
    ↕
每月：抄表录入 → 生成账单 → 收款标记 → 导出账单
    ↕
退租结算 → 归档合同 → 房间恢复空置

全程操作日志  +  生成账单自动备份（本地 JSON + 邮件）
```

## 数据安全

- **本地存储**：所有数据存储在手机本地（uni.storage），不上传服务器
- **JSON 导出**：支持手动导出/导入完整数据备份文件
- **邮件备份**：生成账单时自动通过 Resend 发送加密备份到指定邮箱
- **操作日志**：每次修改都有详细记录，包含楼栋和房间号信息
- **密码保护**：清空数据、导入数据、删除楼栋等危险操作需输入管理密码

```bash
# 3. 构建 → 发行 → 原生 App 云打包（需要 HBuilderX 打包权限）
```

## 邮件备份配置

1. 注册 [Resend](https://resend.com) 获取 API Key
2. 打开 App → 设置 → 邮件备份设置
3. 填入 API Key + 接收邮箱 → 保存
4. 生成账单时自动发送数据备份到邮箱
